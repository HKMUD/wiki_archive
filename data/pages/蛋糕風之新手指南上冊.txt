你左手高舉旗幟，右手拿起擴音器，哽嚥地高呼道：“歡迎，歡迎，熱烈大歡迎！”

Welcome back!!

  * 本指南分為上中下三冊
  * 上冊為入門篇
  * 中冊為宗帥工作篇
  * 下冊為快速通道篇

本指南旨在為大家介紹本mud基本須知練功方向工作機會等，令大家可以快速上手<del>一齊robot</del>



----

本mud比舊hkmud更完善，門派更多，練功速道更快，亦增加了一項終極目標(轉世重生)

現在先介紹一些本mud與舊mud的分別

  * hp -g 可直接顯示武功等級內力等上限
  * 本mud不用洗先天天賦，創造新角時可直接自行分配80點，之後解密可加再加39-40點，中隨機故事可再加
  * 所有師父最多只會教授200lv武功技能(非武功技能如讀書寫字,基本醫術不在此限)
  * 和舊hkmud不同，武功技能達180lv之後，可用yanjiu/research提升lv(e.g. research force 1000)，不用去達摩院
  * 本mud可買魔幻獸(即私人座騎)，可輕鬆騎到不同的市中心及門派(rideto) e.g.rideto dl_shop，想了解更多路線，可用help path
  * 本mud脫離師門武功技能會全失，基本技能變成一百，知識技能則不會被波及
  * 如沒有脫離師門，而是直接叛師拜其他門派，會招來特殊npc追殺(不死只暈)，暈後後果同上，另追加其他門派不會再收你
  * 新增 體會，輸入derive提升武學修養提升research速度，裝備劍類武器後用輸入derive sword-cognize提升劍道修養(轉世前只有華山有用)
  * 可abandon exp 及 技能到指定水平
  * ask npc about all可知道一部份npc會回應的選項
  * pk是有後果
  * 本mud tick time 比較快，可設定 set no_minmsg 和set no_moutmsg，不顯示NPC離開進入你所在房間的信息提示(見help settings)


註:mud中部分help file並不準確，本指南經人手確認可放心參閱

====== 新手上路 ======

----


以舊mud作比較，練功流程為:拜師->pass 18->newbie tomb->hh tomb

本mud流程為:
     - 做mygift quest做到直至叫你拜師
     - 拜雪山學火焰刀(男id only，亦可額外去少林學pfm)
     - 拜師
     - 打通任督二脈，修煉元嬰出世，打通生死玄關
     - 成為大宗師
     - 副本拎武器整個人武器
     - 轉世(60億exp，3600skill lv)
     - 轉世(100億exp，3600skill lv)
     - 轉世(200億exp，3600skill lv)
     - 轉世三後(極限)學習終極轉世技能(3選1 陰陽九轉十二重天(yinyang-shiertian)輪回劍(lunhui-sword)霸刀(badao))

當中須注意事項不少，所以建議花少少時間看完三冊指南

本mud中應該以目標為學習終極轉世技能(好似係)

(其中陰陽九轉十二重天(yinyang-shiertian)要求4個不同門派的技能(丐幫 明教 魔教 段氏)，所以每一世也不能選錯!)


----

第一次轉世經驗要求為60億

轉世後，除獻祭技能，特殊技能基本技能最多只保留200lv，知識技能及其它技能最多只保留500lv

所以第一次目標應該為train skill 到3600-4000lv 轉世

看似遙遠，但本mud練功速度達舊mud十倍以上，兩星期內人物等級升到150lv亦不困難(即約7000lv skill limit)

建議:首先開個dummy入mud睇下各party及public skill的先天及性格要求，之後可以sf正式開始新char!

====== 創造新角色 ======

----





開始新角色過程和舊mud差不多

在register email之後，你要選擇角色性格

不同角色性格有不同能力

  * 光明磊落：具有最大的憤怒值和最大的加怒能力，另外還有更好的抗打擊能力（不容易受傷）。
  * 狡黠多變：使用絕招時忙狀態短一些。
  * 心狠手辣：可以運用憤怒技能。另外命中對手時具有更強的殺傷力。
  * 陰險奸詐：攻擊時命中率更高。
  * 國士無雙：學習武功技能不受性格限制（轉世後獲得）

性格影響你能夠學習的武功技能與宗師技能

很多高級技能除先天要求，亦有性格限制(如華山派獨孤九劍只限狡黠多變的性格)

由於性格在轉世前不能更改，所以要小心選擇

你亦會得到兩個特殊技能（special），除銅皮鐵骨(ironskin)外，其他可另外入手(最多8個) 

(傳說有其他特別speical)

====== 先天天賦 ======

----
__你跳入忘憂池，頓時被一股激流卷了進去。__
----


在選擇角色性格後，可選擇天賦，每項選擇的數值要求必須在13和30之間, 共80點。

技能的先天要求請參考各門派的help file及help skill_gift

如你想要入華山可揀選20膂力22悟性19根骨19身法，輸入washto 20 22 19 19 

輸入完成後，恭喜你正式進入全新的hkmud!!


完成創造角色後，你會到達古村

離開古村的方法為ask lao about here，ask lao about 出村，closeeye

closeeye會得到一些基本技能經驗和潛能，離開新手村後就不能回來，離開前先用一些潛能吧

向西行ask 花伯 about 出村，向花伯打聽有關的消息後

__先選擇去武廟__

====== mygift quest ======

----

離開古村到達武廟後，第一件事就是做mygift quest(在古村內已經可以開始做)

mygift quest其實就是新手quest，任務內容大致上是要求你見一些help file，試用一些cmds，幫助你了解本mud

__更重要的是__mygift quest的獎勵對新手非常有用(如私人座騎和先天丹)

新手建議先完成直到【任務名稱】：完成師門任務

你可輸入mygift查看進度

e.g.

【當前輔助任務如下】

【任務名稱】：了解如何開始玩遊戲
【達成條件】：輸入指令 help start 查看相關資訊
【任務獎勵】：神之祝福戰衣x1、內力+1000



輸入help start後, 任務便完成, 共獲取獎勵

【了解如何開始玩遊戲】任務達成：
-獲得物品：神之祝福戰衣x1
-獲得獎勵內力+1000

======  如意乾坤袋 ======
----
mygift quest 會要求你去到武廟對面製作如意乾坤袋(qiankun bag)

如意乾坤袋是很方便的道具，可用來存取item

儲存上限為100，會員上限為300，可透過ntstore 另外增加上限

輸入summon qiankun 即可召喚

**請注意**

如意乾坤袋是跟id的，角色sf後或轉生後

如意乾坤袋內的item不會消失

使用同一id再註冊新角，將可取回相同的如意乾坤袋

----
====== 座騎 (魔幻獸) ======
----


當你拿到如意乾坤袋後

mygift quest 會要求你創作魔幻獸，即座騎

完成創作後輸入 whistle <魔幻獸 id> 召喚，之後 ride <魔幻獸id>

輸入 rideto 查看目的地，之後 ride dl_shop，就可以購買到全mud最便宜的wisdom set!

此外，你亦可透過輸入 ride rec <代號>，增加一個自定rideto點(只限會員)

----

魔幻獸分為五種

  * 麒麟 -> 氣低於100%時隨機治療及回氣100%
  * 青龍 -> 戰鬥時隨機令對手忙亂
  * 朱雀 -> 戰鬥時隨機降低忙亂時間
  * 白虎 -> 戰鬥時隨機震懾對手，令其暫時無法對你進行攻擊，此效果時間隨機為3-8秒
  * 玄武 -> 戰鬥時隨機保護主人，將對手的攻擊攔下

魔幻獸效果視乎魔幻獸自身等級，想快速升級，可以set share_exp <1-100>，將賺取的輕驗按比例分給魔幻獸
   
個人推薦麒麟，只要有最後一滴血，亦可回滿血(除非不回)，練功其間亦可幫手會回氣

其他相關cmds請見 help warcraft，help ride

====== 食丹提升先天天賦 ======
----

當完成頭幾項mygift quest，直至做到去cs n w，叫你去拜師時 __(男id請先停一停)__

你應該已經有座騎，神之祝福set，天賦分配工具，如意乾坤袋，辟谷符

另外的一堆丹就是用來提升先天天賦的天賦丹

  * 神力丸（shenli wan）
  * 仙丹（xian dan）
  * 洗髓丹（xisui dan）
  * 火紅仙丹（huohong xiandan）

先食以上四種丹最多能提升天天賦到五點，但有一定的幾率失敗

如失敗最好先食魔界佛心果(magic fruit)消除失敗記錄，否則可能會再失敗

輸入hp2 -g查詢，直到四項先天__成功數目為5__

(如果食完後升不到5，可到cs n e買)

__之後__食以下四種丹

  * 破陽無極仙丹(wuji xiandan)
  * 蚩尤誅元仙丹(zhuyuan xiandan)
  * 洗髓再造仙丹(xisui xiandan)
  * 神恩通慧仙丹(tonghui xiandan) 

以上四種丹可把先天提高至__6點__

mygift quest每種送你三粒，有一定的幾率失敗

----

天賦分配工具(gift allocate)，輸入 fen <天賦> <點數數量> 的方式可自由分配4點先天屬性。(如丟失可在cs n e 補購)

e.g. fen 悟性 4

====== 先天天賦解密 ======

----

除食丹之外，可透過其他解密提升先天天賦，詳見help add_gift

  * 男ＩＤ通過上述可獲得 str +2，int +2，con +5，dex +3 總數為120 

  * 女ＩＤ通過上述可獲得 str +2，int +2，con +4，dex +3 總數為119

暫時reborn前最高要求的技能先天天賦要求總數為121

可用指令hp2 -g 查看

mud中不定時出現隨機故事可再加先天，不過比較難中(中左未必成功)

可用nt point買四種不同故事(必定成功)，可盡早早購買，nt point 於下冊詳談


如果想改變先天分配，可去cs n e，buy 洗天賦丹(wash gift) 重新分配先天點數，不用完成全部先天天賦解密已經可用

(另外reborn後可重新解密，先天天賦，更上一層樓)

====== 男id*先天天賦解密 ======

----

男id可比女id多加一點先天根骨(大乘涅磐功加先天身法不限於女id)

方法是到雪山出家，拜鳩摩智學火焰刀，學完200lv會加一先天con，之後向鳩摩智打detach

__請先在鳩摩智處輸入detach(help bai，help detach)__

所有special skill會被廢除，非雪山的也不能幸免，不過知識技能可保存

還俗cmds是huansu

註:因少林技可以讀wuji學習並無門派限制，男id可先拜少林出家學pfm(本mud pfm要先向npc請教才可使用)，detach被廢除技能後，透過讀wuji再次學習後仍可使用(有門派限制除外)

----

女id不能拜入雪山，所以可以直接拜心儀門派

大乘涅磐功可以通過讀無量壽經等(峨嵋藏經閣地上)提先至120lv，讀書寫字高一點才去讀吧

====== 每日簽到 ======

----


每日晚上00:00 前，務必前往cs n n w 的武廟，ask qiandao npc about 簽到

向簽到遊戲管理員 簽到員(qiandao npc) 簽到，領取珍貴的nt point

簽到為ntpoint主要來源

領取到nt point 的數量和你的人物等級有直接關係

每10lv增加10點(e.g. 50-59lv 有50點，60-69lv 有60點)

人物等級和經驗成正比

(請參閱wiki 心得 封印經驗 vs 人物等級)

ntpoint 可用於ntstore 購買非常有用的東西，

(join本mud telegram 可一次領取1000 ntpoint)

====== 拜師 ======

----
第一世建議選一些強勁的門派，如古墓、武當、華山、魔教，可輕鬆在3600lv轉世

拜師後(男id拜雪山加先天除外)，應以學成門派所有技能為目標

本mud並不容許向下拜或拜同輩(有些門派如少林容許拜平輩)，門派最終能拜的大掌門未必擁有門派所有技能

所以如想學齊門派所有技能，在拜師升級請小心確認

(例如:靈鷲宮天山童姥的第子，不能向虛竹手下三十六洞七十二島的武林異士請教的武學)

如果學不能學齊，就可於轉世後再學

學pfm不在此限，達到學習使用pfm要求和擁有足夠門派功績後(見中冊task)，向教授pfm的npc，ask npc about <招式名稱>

招式名稱請見門派各自的help file

----

門派大掌門的skill lv多數為1000lv，但是你只能夠學習到200lv(知識技能則為npc的skill lv)

180lv之後主要是靠research來提升基本技能和特殊內功(如research zixia-shengong 1000)

特殊技能和舊mud一樣多數可以practice/lian，但建議用research

提升research速度是一門highhand學問(help research)

其中一點是相關的技能越多越好，所以學得到的全部都學就不會錯
----
====== 脫離師門與判師 ======
----

如果你想脫離本身的門派，主要有兩個方法

  - 脫離師門：在你師父處輸入detach <id>，你所有特殊武功 及 fmskill 會被廢除(包括非本門武功)，基本武功會減至100lv，所有知識技能和已經習得的pfm則不受影響。脫離師門後不可以再拜回該門派(有少數門派沒有此限制。另外，如果曾經在該門派轉世，則可無損脫離，亦沒有不可重新加入的限制)
  - 判師：直接拜入其他門派。所有技能會保留(暫時)，但你會被判定為判師狀態而不能轉世。此外，你之前的門派會不停派出npc追殺你，如失手被打暈，會如脫離師門失去所有特殊武功，判師狀態亦會解除。此後，大部份npc不過收你為徒。(新門派的上位npc也不會收。)

__
眼見中伏新手為數眾多，特別追加華山拜師篇__

本mud華山分裂為劍、氣二宗。招式相同卻是__兩個不同門派__!

其獨特之處是劍宗可拜__高矮長老__加入氣宗而沒有懲罰(但不可逆轉)

__注意__是拜__高矮長老__! 

千奇千奇，不要拜錯人!

如果你拜了其他人如寧中則勞德諾，則視為叛師，穆人清會找你，而之後亦__不可以__再拜上高矮長老!(或其他人)

----


簡單來說，拜劍宗是為了學pfm(本mud pfm要向指定同門npc請教後方能使用) 及狂風快劍

所以流程應為:

  - 拜劍宗學pfm
  - 拜氣宗高矮長老(不用detach直接拜)

----


劍氣得與失

只拜氣宗: 不能習得 狂風快劍 及 一些華山劍法劍宗pfm (奪命連環三仙,劍天外飛龍,劍芒)

只拜劍宗: 不能習得大量氣宗技能 如紫霞神功 及 華山劍法氣宗pfm

先拜氣宗後拜劍宗: = 只拜劍宗 + 華山劍法氣宗pfm + 叛師

__先拜劍宗後拜氣宗高矮長老: 能習得全所有技能與pfm__


__(獨孤九劍 不受以上影響)__
