====== 大宗師 ======

----


【閑聊】葵花太監(kuihua taijian)：真是長江後浪推前浪，想不到今日又有拿破倫酥躋身武林大宗師的行列！

【閑聊】黃裳(huang shang)：恭喜這位道兄，不過武道永無止境，還望繼續努力啊！

【閑聊】獨孤求敗(dugu qiubai)：這位道兄可要善自珍重，須知強中更有強中手。

【閑聊】南海神尼(nanhai shenni)：不錯，不錯，真是不錯。

【謠言】聽說拿破倫酥成為武學大宗師。


----
要成為大宗師要, 首先要搵四位npc宗師(rideto huang，dugu，nanhai，kuihua)

每位宗師會測試你一種功夫，分別為拳腳，兵器，內功，輕功

ask 宗師 about opinion，之後輸入fight，相方會過一招，然後會給予你評價(可用score查看)

當全部5星, 你會成為大宗師! (5星要求為, 約500 effective skill lv)

如果本身氣量不足，很容易被打至重傷，建議先完成打通任督二脈

----


成為大宗師後，可以跟據性格學習一種終極skill(轉世後四種皆可習得)，可以做宗師任(quest 宗師id，一些npc在領quest後不阻路)...

但主要是指標性作用為主，証明你已經準備好全力robot!


成為大宗師後，跟據性格向一位相應宗師學習一種終極skill(內功輕功招架和其他的基本技能)

      * 光明磊落 九陰神功 (jiuyin-shengong)，其他基本技能: 拳 掌 爪 手 指 拳腳 (九陰神功能enable鞭)

      * 狡黠多變 不敗神功 (never-defeated)，其他基本技能: 刀 劍 棍 杖 鞭 錘 短兵 暗器 拳腳(不敗神功不能enable拳腳)

      * 心狠手辣 南海玄功 (bluesea-force)，其他基本技能: 掌 指 杖

      * 陰險奸詐 葵花魔功 (kuihua-mogong)，其他基本技能: 劍 指 拳腳(葵花魔功不能enable拳腳)

終極skill可以enable做內功輕功招架和和相應的基本技能，威力強勁，更有call die pfm，可直接秒殺低lv npc(玩家)，方便做某些job

但亦同時被多種相應基本技能限制lv，而且只能research，雞肋雞肋

(如九陰神功爪 受 手 拳腳 內功 輕功 招架 限制)

====== 任督二脈 元嬰出世 生死玄關 ======

----

新手來說，就是精和氣不足是一大問題，不單影響練功速度，面對強敵，容易被數招斃命!

打通任督二脈後，可一次性增加大量精氣! 通過宗師測試時如手到拿來!

而且你可以閉關(closed)修煉，方便長時間發呆!(見help closed)

本段落先介紹打通任督二脈(breakup)和修煉元嬰出世(animaout)(何參閱help closed)

(如何在不同store買丹食快速升精力和內力，請參閱新手指南下冊)

----
【謠言】聽說拿破倫酥(napoleon)經過閉關苦修，終於打通了任督二脈，窺視到了無上的武學境界。

----

打通任督二脈的要求為 400lv effective force和一定pot(建議2~300k，一日約消耗100k)

(男id 可去少林拿易筋經學易筋經神功，可讀到380lv (不用pot!!!))

去任可一個室內睡房，輸入 breakup

輸入後除發呆和chat之外，什麼也不能做(就像舊mud被jail)

breakup的時間由數小時到數日不等(通常大半日)

期間如quit，人物會顯示為離線，其他人仍然可以見到你


----

【謠言】聽說拿破倫酥(napoleon)經過閉關苦修，終於修成正果，練得元嬰出世。

----

完成breakup之後，可立即進行元嬰出世(animaout)

過程同breakup一樣，一次性增加大量精和精力

完成animaout之後，可再打通生死玄關(death)，過程同上

完成生死玄關後不過有任可精氣加乘

但會新增死亡後有一定幾率減技能lv，不一定要立即進行，建議夠pot可順手做埋

註：生死玄關時間通常比較長，請耐心發呆

----
【謠言】聽說拿破倫酥(napoleon)經過閉關苦修，終於參悟出生死之道，打通了生死玄關。

====== 人物等級 ======

----
人物等級主要影響工作reward的多少，一些新手job在reward 會隨人物等級減少，一些老手job則相反

當角色經驗上升時，人物等級亦會相應上升。放棄經驗時人物等級__不會下降__。(見wiki 資料庫封印經驗 vs 人物等級)

輸入 hp -g 可查閱一些數值上限(如武功、內力)，其中當前等級就是你的角色等級。

此外，每次等級上升時，你會得到一些ability point，可用於提升ability。(見下冊)

而最重要的是人物等級影響每天簽到能獲得的nt point。(見下冊)

----

===== 拾冰/gleaning =====  

----

在本npc中有一種移動npc，食環清潔工(walker)<del>，會把地上item清走從而降低system負荷</del>

會回收另一種移動npc冰條俠所製造的冰封物品

首先向任何一位食環清潔工學習拾荒技能(gleaning)  

然後向食環清潔工打聽有關『help』的消息

他會要求你在限期前找數件冰封的垃圾

你只需要拾取地上冰封的物品，再交回任何一位食環清潔工即可

請注意冰封的物品在inventry內三分鐘就會溶解，食環清潔工不會接收

建議要求：沒有特別技能要求

  * 限制：已改為三轉限定，新手不能做！
  * 回報：良好經驗，良好潛能，海量體會
  * 其他回報：每二百件增加幫派榮譽點二百點

[新手筍工指數 0/10]

===== 門派防守 =====

----
此job須要有多於100m經驗，回報與經驗成正比，是其中一種最好的高等級工作

門派掌門會定時在門派頻道提醒有敵人來襲和防守結束

此期間向掌門輸入defend <掌門id> 領取工作，門派所在地會出現些敵人npc (skill lv 900lv)

把門派範圍內的敵人殺死(領取工作後會向你auto kill，每殺一位npc會有一定獎勵)

等待任務完成訊息，收到後回掌門輸入finish <掌門id> 領取獎勵

如果掌門望著你呵呵的傻笑，代表門派防守未開始或已經完結

在下一次防守開始時請先領取獎勵，在防守任務開始前先輸入__finish <掌門id>__

建議要求：700lv(npc skill 900lv)

  * 限制:exp >100,000,000
  * 回報:大量經驗，大量潛能，中等門派功績(獎勵隨等級上升增加)
  

[新手筍工指數 3/10] 
[高等級時9/10]

===== 王朝戰爭 kingwar =====

----

===== 使命任務 task =====


----


task 簡單來說就是找尋失物的工作

task 能獲取大量門派功績並且是全要來源，用於學pfm學技及到fmstore買菩提子，明玉神功秘籍等(下冊詳述)

但因為task數目有限，多人做搶，新手不建議做


----
首先輸入task

清單會顯示出76位人名及其失物名稱與id

你須要在使命任務重新分配前尋回失物並交回到失主即可完成

清單上的人物與失物是固定不變(X衣的『賣身契』是例外，八位X衣都試下吧)

npc的位置也是固定，同名的隨便一個也可以 (請見help path)

task 每隔30分鐘重啟一次，重新分配前三分鐘會有以下提示

__【使命精靈】使命任務將於三分鐘後重新分配。__

重啟後所有item會reset，重新分配到不同地方，同時cs，s，e的唐楠會宣佈big task，表示新一回使命任務正式開始!

使命任務共有76件item，簡稱small task 

唐楠宣佈的其中12件item為big task

big task只能夠向唐楠打聽，唐楠會要求你以4件small task item作為交換

交回big task的回報比獨立交回4件small task 更可觀(玩成一次的門派功績足以買菩提子提升數千內力)

正因如此，big task十分搶手

如果4件item 其中一件被搶，將不能完成，所通常以手握第一item者為優先

如果發現item被其他人先找到，可友善地在chat channel求助

===== 搜索 =====

----


當見到米字記號，有兩種可能性
  
1. 在房間的npc(或玩家身上)

ask npc about <task item 中文>

如果在他身上，他會要求你

  - 給他1 gold (無額外reward) -> 家丁對你說道：這位大師如果願意出一兩黃金的話，在下願意把唐家家譜賣給大師。
  - 叫你拿東西交換(有額外reward)-> 親兵對你說道：在下這唐家家譜只換不賣，要換得話，拿粽子來。
  - 用某地方風景畫交換(有額外reward):你必須先去魯班學習50lv drawing，再問魯班拿圖紙，之後去隨便draw here(有bug)後交回npc

時間關係，可以直接落kill，item會在屍體內，如房間是不能戰鬥，請努力滿足npc吧 *giggle

此外，房間的npc可能是動物，只能下kill確認，如是馬，騎去隔壁kill吧

__以上是較好結果，新手也可以完成__

*註:如同房玩家有機會誤拿，因為同一個地方可能多過一個item

*如定位標示改變及移動，可能是玩家已拿了，亦有可能是在會移動的npc身上(多數是拾荒者)

註: task item只會在固定名字的npc身上，不是會在隨機名字的npc 如XX派高手身上




----

2. 在房間環境內

如不在房間npc身上，又或者房間沒有npc

輸入search，你會開始在這裡摸索房間

如fail，__找了半天，你還是一無所獲，只好先放棄了。__，重試數次直至見到以下其中一個結果

  - 直接search到item -> 你突然發現了一本唐家家譜！
  - 進入maze ->祝你好運氣！咦，這裡好象有個隱蔽的迷宮，你撕下封條進入迷宮大門。
  - search到其他普通item如鋼杖長劍(或4-5次仍fail)

如果是第二結果

你會進入maze，必須殺死maze中唯一npc，上古妖王，拿取它身上的item

入maze後向西行一步，輸入mazemap查看自己位置(藍色五角星)，前往紅色方塊

上古妖王就在紅色方塊西方，殺死它後get item from corpse，再west，up，就會回到武廟

如果是第三結果就相對麻煩

因為map的空間重疊了，如cs s w 和 cs w s在位置判定上是一樣的

所以如果search 不成功，嘗試查map看看空間重疊的地方再找找吧

      * 回報:非常良好經驗，非常良好潛能(越高等級越多)
      * 其他回報:少量體會，海量功績
