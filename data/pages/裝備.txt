裝備類型: 三大類別: 武器(刀,劍,鞭,錘,杖,棍,簫(簽名後會變成劍),暗器, 空手武器:指套,掌套) , 首飾(戒指,項鏈,護身符), 防具(護腕,披風,戰衣,甲,束帶,靴,頭盔), 轉生所需的是武器, 新手可以先無視其餘兩類, 注意暗器雖然是武器但不能裝備,也不能用來轉生.

裝備品級: 裝備材料一定是劣質的, 簽名後可 reform 至完美. (詳見[[裝備#改造_reform|第2節]].)

綁定類型: 可以不理

裝備要求: 20或以下的裝備效果會比較低, 建議用30或以上的, 等級越高, 武器傷害力 和 防具保護力 會越高, 但同時需要高些人物等級才可裝備, 如果之後會給其他帳號用的話自行斟酌. 詭異墓園副本的裝備會是30或40. 下面要求的臂力/悟性/根骨/身法亦和要求等級有關. 簽名後可 forging 提升. (作用不大, 不建議使用, 詳見[[裝備#鍛造_forging|第8節]].)

耐 久 值: 使用武器類攻擊時會扣耐久值, 扣到零後不能使用, 需要到鐵匠(rideto yangzhou,e,e,s)處 repair. 如不想扣耐久值可以 cuilian. (詳見[[裝備#淬煉_cuilian|第3節]].)

裝備效果: 武器有武器傷害力(空手武器另有10點保護力), 防具有防具保護力, 首飾固定10點保護力, 另外視裝備品質會有一個(珍稀)至四個(神器)其他效果.

=====2. 改造(reform)=====

這是用來提升裝備品級的指令, 裝備品級依次為 劣質的(1), 普通的(2), 精緻的(3), 卓越的(4), 無暇的(5), 和 完美的(6), 括號內數字為裝備對應的status. 要轉生必需一把完美的武器, 即是要成功 reform 五次. 

reform 要準備三種材料: 1.裝備之魂, 2.寰宇天晶(tian jing) 或 無名之物(no name), 3.歐冶子聖符(ouyezi symbol)

其中 1,2 為必要物品, 3 是用來提高成功率的 (可以不用, 不過一般都會買來用).

1.  裝備之魂(xxx soul<no.>),  xxx為裝備類型(weapon,finger,hand,charm,armor, etc), <no.>標示等級,如要升武器成為無暇的(status 5)最少要用 weapon soul5.

寫在前面: 

上面提過打神器裝需要 600%以上尋寶率 (magic_find), 而要達成目標只有一個方法: 能力水晶(ntstore 151, 詳情請線上問高手), 過程中裝備的鑲嵌物及槽數會不斷被覆寫, 所以如果有打算儲 NT 打神器的話自行斟酌要不要前期花資源鑲嵌.

在戰鬥中經過鑲嵌的武器**(經實測發現指套有bug, 沒有效果, 想有鑲嵌效果的話不要用指套)**打中對手會有以下一行:

  你的「xxxx」發出一道奇異的光芒！

按機率觸發以下效果 (似乎只計鑲嵌效果, 裝備本身或強化的數值並不會計算在內, 如果裝備本身有下列效果其實是沒有作用的):

  偷取生命 (suck_qi): 額外偷取攻擊造成傷害的 10% 氣血 
  偷取內力 (suck_neili): 額外偷取攻擊造成傷害的 10% 內力 (似乎內力比最大內力值多的話會變成最大內力值, 即不能 db fp) 
  致盲 (add_blind): 對方會看不到訊息 (即對 npc 基本上沒有任何影響)
  冰凍 (add_freeze): 對方短時間(數秒)內變成冰凍狀態, 影響招架及會 perform 
  眩暈 (add_dizziness): 清除戰鬥狀態, 對方短時間(數秒)內變成眩暈狀態, 似乎會影響 alias  
  遺忘 (add_forget): 對方短時間(數秒)內變成遺忘狀態, 不能使用 perform 
  破氣 (clear_force): 使對方現內力值歸零
  必殺率 (fatal_blow): 有 20% 機會直接扣除對方對應百分點的氣血. 
  天魔附體 (absorb_blood): 增加三成攻防數值 (似乎並不會回復氣血)
  殺戮 (slaughter): 對同一地方正在與你生死相搏的生物造成少許傷害.

鑲嵌需要 鑲嵌技藝 (certosina), 可以跟干將學, 最少要 200 級, 400 級以上不會鑲嵌失敗. 學完可以在任何地方鑲嵌. 

要預備的東西只有 一件有空槽的裝備(不限簽名裝備) 及 要鑲嵌的 寶石/符文.

上面第4節提過如何開槽, 如果要清空槽位可用指令 melt xxxx (只限簽名裝備, 清空所有槽位) 或 附魔(enchant 詳見[[裝備#附魔_enchant|第9節]]).

可用來鑲嵌的東西主要有 寶石 及 符文 兩種. 符文可以參看 help rune. 以下只介紹寶石.

寶石主要有兩種, 打 boss 掉落的五色寶石 及 日魂月魂 (附魔的產物, 詳見[[裝備#附魔_enchant|第9節]]).

五色寶石為 藍寶石(sapphire) 紅寶石(ruby) 水晶鑽(diamond) 翡翠綠(emerald) 黃寶石(topaz)

不同顏色對應五行屬性, 依次為水火金木土. 有屬性的完美武器打中對手會有扣精/氣的特效 (不過其實作用很小). 留意同一件武器只可以鑲嵌同一顏色的寶石.

土屬性例子:

  xxxx急速的劃過土石之緣，帶動山石之雨，咆哮著令人心膽俱裂的震天怒響擊向xxxx。 

每種寶石有四個等級, 以紅寶石為例: 細碎的紅寶石(chipped ruby), 紅寶石(ruby), 稀世紅寶石(flawless ruby), 神之紅寶石(magic ruby).

各級主要分別為附加特性的數目(由 1個 到 4個, 其中一個必為屬性傷害(武器/飾物) 或屬性抗性(防具)). boss 掉落的寶石為未鑒定寶石, 不能直接鑲嵌, 要先找張天師鑒定(give 5 gold to zhang, show xxx). 鑒定後即可知道鑲嵌後的附加屬性, 以下為一例子:

  物品屬性 : 鑲嵌物<神之紅寶石>，重量20克，下線不丟失
  魔力屬性 : 火
  魔 力 值 : 500
  可鑲裝備 : 所有類型(all)
  鑲嵌兵器 : 遺忘 +3%
  鑲嵌兵器 : 根骨 +3
  鑲嵌兵器 : 附加傷害 +1200
  鑲嵌兵器 : 火傷害 +3%
  鑲嵌防具 : 讀書次數 +15
  鑲嵌防具 : 絕招防御力 +3%
  鑲嵌防具 : 抗金 +1%
  鑲嵌防具 : 練習次數 +15
  鑲嵌飾品 : 忽視致盲 +1%
  鑲嵌飾品 : 火傷害 +3%
  鑲嵌飾品 : 內力最大值 +10000
  鑲嵌飾品 : 攻擊無視內功防御 +3%

正如描述, 鑲嵌在不同種類的裝備有不同的附加屬性. 留意不同裝備種類的附加屬性是由不同的屬性池中隨機得出的, 例如 絕招防御力 只會在防具的附加屬性中出現. 詳細可以參考 help element (沒有逐項比對, 不過應該沒有錯). 

另外寶石是可以升級的, 有三粒已鑒定的同色同級寶石可以合成為一粒高一級的未鑒定寶石. (最高級的神之寶石不能再合成)

例子: 

cmd: combine flawless diamond 1 & combine flawless diamond 2 & combine flawless diamond 3

  你雙目微閉，將數樣物品凝於掌中，運轉內勁迫使它們交融。
  霎時間只見萬道光華疾閃而過，你急忙攤開手掌，發現掌心
  豁然出現了一顆神之水晶鑽，蘊漾著七色霞光。

=====6. 強化(qianghua)=====

  你向干將打聽有關『強化』的消息。
  干將對你說道：想要知道強化裝備需要的材料，請show <裝備>，我會告訴你所需的材料。

cmd: show <裝備>

  你拿出一件xxxx對干將說：“拜托拜托...”
  干將說道：強化xxxx到1星需要材料有：乾坤聖水、符文rune28 3個以及下列的屬性元素
  汲取效果元素3個、化解忙亂元素3個、研究效果元素3個，
  如果你備齊了以上材料，請輸入qianghua <裝備>進行強化。

乾坤聖水 (ntstore: 152), 符文 3個 (升 1星 rune28, 升 2星 rune29, 如此類推直至 升 9星 rune36) 為所有強化必要物品.

其他元素 (ntstore: 147-150, 詳細可參看 help element) 則視裝備的**所有**鑲嵌附加屬性而定, 各需 3個, 亦是必要物品.

部分元素可以打九頭蛇精(留意**不是** ntstore 15-17 那個, 要深入神龍島某洞穴找) 和附近的火麒麟得到, 詳細請參看 help boss (雖然有些資料是錯的).

集齊即可強化, 成功率如下:

^ 升   ^ 成功率  ^
| 1星  | 70%  |
| 2星  | 70%  |
| 3星  | 70%  |
| 4星  | 50%  |
| 5星  | 35%  |
| 6星  | 20%  |
| 7星  | 15%  |
| 8星  | 10%  |
| 9星  | 5%   |

如裝備以無名之物淬煉(cuilian, 詳見[[裝備#淬煉_cuilian|第3節]]), 各成功率加 5%.

升至 6星為止用強化水晶 (ntstore: 154) 可以確保成功. 如裝備是 5星以上, 升星失敗而又沒有聖靈水晶 (ntstore: 155) 在身上的話裝備會跌回 4星.

cmd: qianghua <裝備> with crystal **(銀行會扣一萬金存款.)**

  干將(gan jiang)說道：不錯，既然我要的東西你都備齊了，我這就開始強化xxxx。
  
  
  干將將乾坤聖水、符文和元素粉末混合在一起融合，然後將融合後的乾坤聖水緩緩洒在xxxx上，
  頃刻間xxxx猶如烈火中重生一般，散發出絢目的光彩！
  
  強化成功！xxxx強化等升至 1 星。
  你感受xxxx發生了不可言喻的變化。
  干將拿出xxxx還給你。
  
強化後所加的屬性與裝備的鑲嵌屬性(即消耗了的元素項目)是一樣的, 而所強化的數值是根據星數固定的. (例: 鑲嵌效果: 汲取效果 +3% 與 汲取效果 +100% 的同星強化效果是一樣的.) 詳細數值可參看 code (/adm/daemons/equipmentd.c 裡的 steady_values).

強化水晶和聖靈水晶都是貴價材料, 雖然其他元素符文單價不太高, 但數量不少, 升星每次要花數百甚至上千 NT, 加上強化數值不算太高, 請自行衡量要不要花資源強化, 尤其是成功率低的升 7星 至 9星.

注意強化後用 melt 指令會重置成零星.
