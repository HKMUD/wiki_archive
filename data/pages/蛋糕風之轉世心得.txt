====== 蛋糕風之轉世心得 ======

----
====== 轉世 ======

----


轉世重生可分為上下兩篇，分別為五嶽獻祭篇及地府輪迴篇

由五嶽獻祭篇開始，玩家須到揚州城武廟向南賢拿取閻王契，到五嶽挑戰五嶽府君的測試並作獻祭以完成簽署閻王契。

之後經昆侖之巔的昆侖鏡進入地府開始地府輪迴篇，先幫助十殿閻羅尋找鎖陽丹的四種原料，然後再挑戰六道守護者！

挑戰成功後你將會獲得特定的六道輪回先天技能

最後得三界法王釋迦牟尼指點重生轉世後，使用黃樑枕與佛祖再續前緣，你的轉世重生就正式完成！

----


轉世重生完成後，佛祖會恢復你前世的武功、ntpoint、召喚裝備、前世座騎、如意乾坤袋等

﹝武學類保留200 級，知識類保留500級，獻祭技能的等級則保留獻祭的等級減3000﹞

----


前生special和inborn會取代現有記錄，__使用黃樑枕之前切勿做任何增加先天的quest或食先天丹__

﹝先天quest增加的inborn會被上一世記錄取代，先天丹增加的inborn則會被扣除，而記錄上則顯示為完成﹞

----


另外，你可以有重新選擇新門派來練功同時可無損失脫離前世的門派一次。

在拜回前世門派時，你只需要拜任何一位門派的npc

而之後你就可以用skill/cha來查看所有門派npc/玩家的技能，和不受限制下學習他們的所有技能(有少數技能不能學)直到你脫離師門！

脫離師門只需向師父打detach，即可無損脫離

====== 如何知道自己已經準備好轉世？ ======

----


可先到「北嶽」恆山和恆山府君幻象fight

看看能不能躲避他的calldie pfm...可先打掉他手上武器增加躲避機會

5件刑天套裝也是基本

(我使用技能組合-華山A通關，獨孤九劍3468lv及其他3601lv，配合刑天套裝，不過打六道守護者非常吃力，最後是靠flood auto pfm碌過...)

(之後用差不多lv的圓月刀法...四下pfm了結六道守護者...)
----

拿取「閻王契」的必須要求：

  * EXP 6,000,000,000
  * 基本內功、基本輕功、基本招架、一門空手及一門兵器武功等級達到3600lv (激發的特殊武功達到3600lv即可，裝備提升的技能等級不計)
  * 武學修養 3600lv
  * 一把自己簽名完美品質的兵器
  * 成為武學大宗師，打通任督二脈，修煉出元嬰出世，打通生死玄關

你亦必須了斷塵世瓜葛：

  * 離婚
  * 解除結義
  * 脫離幫派
  * 脫離同盟
  * 拆屋
  * 叛師者需要停止被追殺(即是要被廢除武功)

*註：轉世重生後，能力點、功績、軍功會重置，請先使用！

此外，你亦需要準備祭品，在五嶽獻祭篇中，獻祭給五嶽府君以完成閻王契！

  * 召神香 最少5支 (ntstore buy 51)
  * 3000lv 特殊空手武功 X1
  * 2000lv 特殊兵器武功 X1
  * 2000lv 特殊內功 X1
  * 2000lv 特殊輕功 X1
  * 1,000,000 內力

*註：獻祭的武功等級及內力會被即時扣減

祭品外你亦需要準備一些東西作解迷之用:

  * 銀行存款<del>10,000 至</del> 1,000,000兩黃金
  * 3000lv 特殊空手及兵器武功 約10-20門 (視乎運氣，少至一門也可)
  * calldie pfm X1
  * 強勁武功 X1 (如沒有，建議over 4000lv 才挑戰)
  * 大量天師護身符
  * 一定數學常識或一位數學老師

====== 五嶽獻祭篇 ======

----
你向南賢打聽有關『轉世』的消息。

南賢嘆道：「人間本是百樁磨，既然是有心之士相詢，老夫也不藏拙，這物事你便收下吧。」

南賢拿出一份閻王契(Contract)給你。

南賢續道：「欲成轉世奇緣，需向五岳山君獻祭，你這便前往泰山、華山、衡山、恆山與嵩山一行吧。」

【轉世重生】人間：拿破倫酥(cakei)拿到閻王契，開始前往五岳獻祭，欲成轉世奇緣！

----

轉世重生第一步是去拿取閻王契，__激發正確武功和裝備好正確兵器後__向南賢查獲轉世，你身上就會多了一份空白的閻王契

按自己喜好次序，到以上五嶽祭壇

以召神香召喚五嶽府君(put magic xiang in xiang lu)

查詢獻祭(ask fujun offer)

完成衪們的挑戰並作獻祭之後，獻祭時間及內容會由各府君寫入閻王契

完成五嶽獻祭後，帶同閻王契回南賢處簽名，即可經昆侖鏡前往地府！

====== 地府輪迴篇 ======
----

地府輪迴篇分兩份：

  - 幫助十殿閻羅收集鎖陽丹的材料
  - 打到六道守護佛

完成後即可向地藏王打聽轉世重生，衪就會送你上天界直接面見佛祖！

----

在昆倫之顛進入地府後，你會先到達幽冥殿。你可以先完成十殿閻羅的任務，只要小心行事，任務的難度其實十分低。

在幽冥殿的南方盡頭孟婆處，可先用ride rec <代號> 記錄騎馬點﹝此處亦是地府唯一能summon qiankun 的地方﹞。萬一不小心返回陽間，亦可用rideto返回

如果在地府死亡，你會如常被送到白無常。白無常會送你回幽冥界﹝你的屍體也在此處﹞。如果在白無常處各北行一格找黑無常，衪會送你回揚州武廟。

完成十殿閻羅的任務，衪就會隨機送去其中一位六道守護佛。你只需打倒其中一位即可。

通過打倒不同的六道守護佛，會得到相應的轉世特技。

本mud 暫時只開放三次轉生，所以每人最多只能擁有三項轉世特技，請小心選擇！

詳見help rb_special

註：首次轉世推薦經人道佛，因為經人道轉世能獲得轉世特技智周萬物，可提高每次學習、練習及研究次數上限八百次，學習、練習及研究效果提升50%。全面提升往後練功速度！


在打到六道守護佛後向地藏王打聽轉世重生前，就是儲存身上不能summon的item和處理fm point、ability point 等的最後機會！

最後，當向釋迦牟尼打聽轉世重生後，衪會告訴你一句揭語

__之後就會disconnect，緊記take log，否則失去揭語，就前功盡費！__

四神獸很容易殺，任務難度主要來自解迷，要先拿到必要道具才可安全通關，小心即可

==== 朱雀 ====

----
位置：閻羅殿以西陰陽塔塔頂

必要道具：七色瓔珞

先在到幽冥殿南面的幽冥界看石碑﹝look shibei﹞直至「魂王」秦廣王出現

殺死衪後就能在衪身上拿到七色瓔珞

前往陰陽塔塔頂﹝殺掉阻路的馬面﹞，把七色瓔珞放在供奉台﹝put qise yinluo﹞，即可招引朱雀

----
==== 四神獸之首 ─ 青龍 ====

----
位置：碧水寒潭的青龍洞

必要道具：地陰血脈

首先去寂滅司以東的輪回道，先看對聯﹝look duilian﹞，打開銅門再進入輪回司，之後高聲說出下聯﹝say 下聯﹞，青石桌案會露出一個石制輪盤

轉動輪盤﹝turn lunpan left/right﹞，三左三右，就能打開階梯前往地陰通道，一直下行到盡頭就會到達地陰

在地陰敲一下地陰血脈﹝qiao jingti﹞，之後__從地上拿起一塊地陰血脈__

拿地陰血脈後原路返回幽冥界，在幽冥界南面的奈河橋跳橋﹝tiao qiao﹞，即可進入碧水寒潭

碧水寒潭是迷宮，請用maphere來尋找黃色的房間就能找到青龍洞

在青龍洞搜尋青龍﹝xun﹞直到青龍現身

解決青龍後進入旋渦﹝enter xuanwo﹞就能回到奈河橋孟婆處

此時精氣和氣血會變成0%需要食藥回復，__千萬不要錯吃龍膽__

回幽冥殿有馬面阻路殺掉即可

即是在衪calldie你之前打贏
