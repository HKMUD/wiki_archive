===== 門派簡介 =====
攻擊力:低
防禦力:高

逍遙派並不是一個打鬥強的門派。選擇逍遙派通常是為了兩個目的 - 最強吸內力內功北冥神功和最強輕功凌波微步。下面會再作詳細介紹。

===== 天賦要求 =====
|      | str  | int  | con  | dex  |
| 北冥神功  |      | 30   | 24   |      |
| 六陽掌   | 30   |      |      |      |
| 凌波微步  |      |      |      | 21   |

本mud隨時可以在cs,n,e花1 gold買wash gift洗天賦。
===== 拜師流程 =====
新手拜師逍遙派先從函谷八友拜起(rideto xiaoyao)，學到一些基本武功後就可以拜蘇星河(rideto xiaoyao;w)。學到小無相功有相當功力後就可以去拜逍遙子(從蘇星河處enter;out)。拜到逍遙子就可以學習北冥神功啦！

===== 攻擊技能 =====
六陽掌和折梅手可以互備。普通攻擊如果沒有刑天裝備的話會很無力。建議洗出刑天魔掌或刑天魔指方便做job。
perform方面，折梅手的perform hua或六陽掌的perform po可以秒殺最大內力低於你一半的對手。由於北冥神功可以輕易limit fp，對一些弱小的npc有一定作用。而折梅手的perform hai算是本門派中最強的單招perform，單份氣修害，2 round自身忙亂，可惜威力也只是普普通通。如果覺得威力不足的話可以考慮公開武功。

逍遙派的武器技能威力強差人意，學習的主要原因只是為了過宗師挑戰和提升研究速度，初期可以無視。

===== 北冥神功 =====
全mud最強吸內力內功，當其他門派要不斷嗑菩提子升最大內力，逍遙派弟子依靠北冥神功就可輕鬆limit內力。另外，北冥神功也有很高的內力加成，也就是說同等級下，北冥神功的最大內力比其他的內功高。

skill beiming-shengong可以看到北冥神功所有的exert。不要被exert maxsuck誤導了，maxsuck的公式存在明顯的問題，根本不能吸到內力。吸內力的正確exert是exert suck，吸取到的內力公式是

1 + (北冥神功等級 - 90) / 10

可以找 練功法寶 萬年海龜 慢慢吸(rideto haigui)。

內功特殊防禦（內防）方面，內防是用來抵禦UNARMED/WEAPON類型的絕招加力傷害。北冥神功內防屬於Ｂ級(本mud沒有Ａ級的內防武功)，比較的是雙方的有效內功和有效輕功等級，算是不錯的內防。

===== 凌波微步 =====
全mud最強輕功凌波微步。凌波微步強大的是輕功特殊防禦（輕防），輕防是用來抵禦UNARMED/WEAPON類型的攻擊以及普通攻擊。凌波微步輕防在本mud中是最頂尖的Ａ級，公式如下：

ap (attack power, 攻擊方) = 有效輕功等級 + 陰陽八卦等級 (桃花島專屬知識技能)

dp (defense power, 防禦方) = 基本輕功等級 * 3/5 + 凌波微步等級

if (ap/3+random(ap)<dp) => 傷害無效
