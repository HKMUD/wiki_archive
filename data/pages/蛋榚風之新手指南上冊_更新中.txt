====== 新手上路 ======

建議:首先開個dummy入mud睇下各party及public skill的先天及性格要求, 之後可以sf正式開始新char! 


====== 創造新角色 ======




開始新角色過程和舊mud差不多

在register email之後, 你要選擇角色性格

不同角色性格有不同能力

  * 光明磊落：具有最大的憤怒值和最大的加怒能力，另外還有更好的抗打擊能力（不容易受傷）。
  * 狡黠多變：使用絕招時忙狀態短一些。
  * 心狠手辣：可以運用憤怒技能。另外命中對手時具有更強的殺傷力。
  * 陰險奸詐：攻擊時命中率更高。

但最主要的, 還是影響你能夠學習的武功技能與宗師技能

很多高級技能除先天要求, 亦有性格限制(如華山派獨孤九劍只限狡黠多變的性格)

由於性格不能更改, 所以要小心選擇


====== 先天天賦 ======



在選擇角色性格後, 可選擇天賦, 每項選擇的數值要求必須在13和30之間, 共80點。

技能的先天要求請參考各門派的help file及help skill_gift

如你想要 20膂力22悟性19根骨19身法, 輸入washto 20 22 19 19

輸入完成後, 恭喜你正式進入全新的hkmud!!

__你大聲說：恭喜！恭喜！__

完成創造角色後, 你會到達古村, 強烈建議周圍行一下或和村長溝通一下:) 離開新手村後就不能回來

離開古村的方法為ask lao about here,ask lao about 出村, closeeye, 向西行ask 花伯 about 出村

====== mygift quest ======


離開古村後, 第一件事就是做mygift quest

mygift quest其實就是新手quest, 任務內容大致上是要求你見一些help file, 試用一些cmds, 幫助你了解本mud

而且mygift quest的獎勵對新手非常有用, 所以建議先完成直到【任務名稱】：完成師門任務

你可輸入mygift查看進度

e.g.

【當前輔助任務如下】
------------------------------------------------------------------
【任務名稱】：了解如何開始玩遊戲
【達成條件】：輸入指令 help start 查看相關資訊
【任務獎勵】：神之祝福戰衣x1、內力+1000
------------------------------------------------------------------

輸入help start後, 任務便完成, 共獲取獎勵

【了解如何開始玩遊戲】任務達成：
-獲得物品：神之祝福戰衣x1
-獲得獎勵內力+1000



食丹提升先天天賦

當完成頭幾項mygift quest, 直至做到去cs n w, 叫你去bai party時 __(男id先不要去bai party)__
你應該已經有座騎, 神之祝福set, 天賦分配工具, 如意乾坤袋, 辟谷符, 和一堆丹

天賦分配工具(gift allocate), 用fen (天賦) 的方式可自由分配4點先天屬性。

e.g. fen 悟性 4


另外的一堆丹就是用來提升先天天賦的天賦丹

  * 神力丸（shenli wan）升str、仙丹（xian dan）升int、洗髓丹（xisui dan）升con、火紅仙丹（huohong xiandan）升dex

先食以上四種丹, 每種丹能提升五點先天天賦, 但有一定的幾率失敗, 如失敗最好先食魔界佛心果(magic fruit)消除失敗記錄, 否則可能會再失敗

輸入hp2 -g查詢, 直到四項先天成功數目為5

如果食完後升不到5, 可到cs n e買, 或;

  * 破陽無極仙丹(wuji xiandan)蚩尤誅元仙丹(zhuyuan xiandan)洗髓再造仙丹(xisui xiandan)神恩通慧仙丹(tonghui xiandan) 

以上四種丹是用來提高第六點天賦(理論上), 必定成功

但實際上, 沒有限制, mygift quest每種送你三粒, 所以之如升了三點已經ok

====== 先天天賦解密 ======

食丹之後, 可透過解密提升先天天賦, 詳見help add_gift


男ＩＤ通過上述可獲得 str +2, int +2, con +5, dex +3; 總數為120 
女ＩＤ通過上述可獲得 str +2, int +2, con +4, dex +3; 總數為119


另外, mud中隨機故事可再加4點, 不過比較難中

如果想改變先天分配, 可去cs n e, buy 洗天賦丹(wash gift) 重新分配先天點數(不用完成全部先天天賦解密可用)

====== 續*先天天賦解密 ======

男id可比女id多加一點先天先天根骨(大乘涅磐功加先天身法不限於女id)

方法是出家拜入雪山學火焰刀(鳩摩智), 出家學完200lv之會加一先天con, 之後向鳩摩智打detach

本mud脫離師門是有後果的 (help bai, help detach), 還俗cmds 是huansu

女id可以直接拜心儀門派不用bai emei, 可以等讀書寫字高lv一點才去emei拎shu讀

====== 拜師 ======

拜師後(男id拜雪山加先天除外), 應以學成門派所有技能為目標

本mud並不容許向下拜或拜同輩, 門派最終能拜的大掌門未必擁有門派所有技能, 

(例如:華山第十三代是不能拜寧中則學習玉女金針十三劍)

所以請小心確認

學pfm則不在此限

(本mud pfm 多數須要學習才能使用, 達到使用pfm要求和擁有功績後, 向教授pfm 的npc, ask npc about <pfm 名稱>)

門派大掌門的skill lv 為1000lv, 但是你只能夠學習到200lv(知識技能不在此限)

200lv之後主要是靠research來提升基本技能和特殊內功, 特殊技能和舊mud一樣多數可以practice/lian

提升research速度是一門highhand學問(help research)

其中一點是相關的技能越多越好, 所是學得到的全部都學就不會錯
